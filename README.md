# Kobo 中國象棋 / Kobo Chinese Chess (Xiangqi)

Kobo e-ink 中國象棋遊戲，單一 HTML 檔案，無需安裝。
Chinese Chess for Kobo e-ink readers. Single HTML file, no install needed.

## 功能 / Features

- 完整象棋規則（將士相馬車炮兵）/ Full Xiangqi rules
- AI 對手（minimax + alpha-beta，難度 1-3）/ AI opponent (depth 1-3)
- 悔棋、新局 / Undo, new game
- 為 e-ink 灰階螢幕最佳化 / Optimized for e-ink grayscale display
- 純 ES5，零外部依賴，完全離線可用 / ES5 only, zero dependencies, fully offline
